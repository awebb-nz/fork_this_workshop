# Fork This Workshop

An example repository for a git workshop.

## How to Contribute

Follow these steps to add yourself to the People folder:

1. **Fork this repository** by clicking the "Fork" button at the top right of this page on GitHub.

2. **Clone your fork** to your local machine:
   ```
   git clone https://github.com/YOUR-USERNAME/fork_this_workshop.git
   ```

3. **Create a new file** in the `People/` folder named `<your-github-username>.md`. Add a short description of yourself (or a made-up character if you prefer!).

4. **Commit and push** your changes to your fork:
   ```
   git add People/<your-github-username>.md
   git commit -m "Add <your-github-username> to People"
   git push origin main
   ```

5. **Open a pull request** back to this repository by going to your fork on GitHub and clicking "Contribute" → "Open pull request".

That's it — welcome to the workshop!
